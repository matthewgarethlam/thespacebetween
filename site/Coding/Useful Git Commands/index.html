Useful Git Commands

Setup

Initialise a new repository:

git init

Clone an existing repository:

git clone <repository-url>

Check your Git version:

git --version

Commonly Used

Check repository status:

git status

See recent commits:

git log

Add all changed files:

git add .

Add a specific file:

git add myfile.py

Create a commit:

git commit -m "Add feature X"

Branching

View branches:

git branch

Create a new branch:

git checkout -b feature/new-feature

Switch branches:

git checkout develop

Merge a branch:

git merge feature/new-feature

Delete a branch:

git branch -d feature/new-feature

Pushing and Pulling

Download latest changes:

git pull

Upload commits:

git push

Push a new branch:

git push -u origin feature/new-feature

Comparing Changes

Compare branches:

git diff main develop

Undoing Mistakes

Undo changes to a file:

git checkout -- myfile.py

Unstage a file:

git reset myfile.py

Revert a commit safely:

git revert <commit-id>

The "Save My Work Quickly" Commands

Temporarily save changes:

git stash

Restore stashed changes:

git stash pop

"Do the dance"

This is my favourite. Used when I need to update branch without losing local changes: 

git stash #this stashes the changes in my current branch
git checkout main #checks out the main branch
git pull 
git checkout myoriginalbranch
git stash pop 
git merge main #brings in all the changes from main into current branch
